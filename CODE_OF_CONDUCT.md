# 行为准则

## 我们的承诺

为了营造一个开放、友好的社区环境，我们承诺：尊重每一位参与者，无论其经验水平、性别、身份背景或信仰。

## 我们的标准

### 积极行为

- 使用友好、包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 关注对社区最有利的事情

### 不可接受的行为

- 使用侮辱性/歧视性语言或人身攻击
- 公开或私下的骚扰行为
- 未经同意发布他人的私人信息
- 其他被认为不专业的行为

## 执行

项目维护者有权利和责任删除、编辑或拒绝不符合本准则的评论、提交、代码、Issue 和其他贡献。维护者可暂时或永久禁止任何有不恰当、威胁性、冒犯性或有害行为的贡献者。

## 适用范围

本准则适用于所有项目空间，也适用于个人在公开场合代表本项目时的行为。
