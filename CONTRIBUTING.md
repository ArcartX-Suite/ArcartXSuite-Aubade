# 贡献指南

感谢你对 Aubade 项目的关注！

## 如何贡献

### 报告 Bug

1. 先搜索已有 Issue，避免重复报告
2. 使用 Bug 报告模板，提供复现步骤和版本信息
3. 附上相关日志片段

### 提交功能建议

1. 使用功能建议模板
2. 描述使用场景和预期行为
3. 如有实现思路，欢迎一并提出

### 提交代码

1. Fork 本仓库并创建特性分支 (`git checkout -b feature/xxx`)
2. 遵循现有代码风格，注释使用中文
3. 确保 `./gradlew build` 编译通过
4. 提交 Pull Request，填写 PR 模板

## 开发规范

- **Java 版本**：Java 21
- **构建工具**：Gradle (Kotlin DSL)
- **代码风格**：保持与现有代码一致，优先简洁
- **注释语言**：中文
- **API 兼容性**：目标 Paper 1.20.1，通过运行时反射适配高版本

## 联系我们

如有问题，欢迎通过 GitHub Issue 讨论。
